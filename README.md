# Налоговый навигатор ИП 2026

Калькулятор сравнения налоговых режимов ИП, портированный из Excel в **настоящее приложение**:
продающий лендинг + Telegram Mini App с freemium-монетизацией.

Считает 6 режимов на 2026 год с учётом реформы НДС: **НПД, УСН «Доходы» 6%, УСН «Доходы-Расходы» 15%, ПСН, АУСН 8%, АУСН 20%** — и подсвечивает самый выгодный.

---

## Что внутри

```
tax-navigator/
├── shared/            ← ОБЩИЙ движок (одна логика для сайта и мини-аппа)
│   ├── params.js      ← все ставки и лимиты 2026 (правится раз в год)
│   ├── engine.js      ← расчёт всех режимов + точки перелома
│   └── format.js      ← форматирование рублей и процентов
├── landing/           ← продающий сайт с живым демо-калькулятором
│   ├── index.html
│   ├── styles.css
│   └── app.js
├── webapp/            ← Telegram Mini App (free + Pro)
│   ├── index.html
│   ├── styles.css     ← подстраивается под тему Telegram (свет/тьма)
│   └── app.js         ← free-расчёт + Pro-разбор + PDF + оплата
├── bot/               ← бэкенд: оплата Telegram Stars и выдача Pro
│   ├── server.js
│   └── package.json
└── test/
    └── verify.js      ← проверка движка по эталонам из Excel (20 тестов)
```

**Ключевая идея:** вся налоговая логика живёт в `shared/engine.js`. Лендинг и Mini App
её просто импортируют — расчёт нигде не дублируется, обновлять надо одно место.

## Freemium-модель

| | Бесплатно | Pro (1490 ₽ / год) |
|---|---|---|
| Сравнение 6 режимов | ✅ | ✅ |
| Лучший режим + экономия | ✅ | ✅ |
| Детальная разбивка (налог/взносы/НДС) | — | ✅ |
| Сценарии роста на 2 года | — | ✅ |
| Точки перелома | — | ✅ |
| PDF-отчёт | — | ✅ |

Pro — **годовая подписка 1490 ₽/год** (≈124 ₽/мес). Модель выбрана под ритм продукта:
налоги пересматривают раз в год, и раз в год обновляются все ставки/лимиты — есть за что платить снова.
Бот выдаёт доступ на `PRO_DAYS` дней (по умолчанию 365) и хранит дату окончания у каждого пользователя.

Оплата — через **Telegram Stars** (валюта XTR). Не нужен эквайринг, договор с банком
и проверка статуса ИП: Telegram берёт расчёты на себя. Позже легко перейти на рубли (ЮKassa).

## Как запустить локально

### 1. Лендинг и Mini App (статика)
```powershell
cd tax-navigator
python -m http.server 8124
```
- Лендинг:  http://localhost:8124/landing/index.html
- Mini App: http://localhost:8124/webapp/index.html  (в браузере Pro можно включить через `?pro=1`)

### 2. Проверить движок расчёта
```powershell
node test/verify.js
```
Должно быть `✅ Итог: 20 прошло, 0 провалено` — значит логика совпадает с исходным Excel.

## Как выложить в Telegram (по шагам)

1. **Создать бота.** Напишите [@BotFather](https://t.me/BotFather) → `/newbot` → получите `BOT_TOKEN`.
2. **Выложить статику.** Папки `webapp/` и `shared/` — на любой HTTPS-хостинг
   (Netlify, Vercel, GitHub Pages, Cloudflare Pages). Нужен именно **HTTPS** — Telegram иначе не откроет.
   Важно: `webapp/index.html` импортирует `../shared/`, поэтому выкладывайте обе папки рядом,
   сохранив структуру (`/webapp/…` и `/shared/…` на одном домене).
3. **Привязать Mini App.** В @BotFather: `/newapp` → выберите бота → укажите URL `https://ВАШ_ДОМЕН/webapp/index.html`.
4. **Запустить бэкенд оплаты:**
   ```bash
   cd bot
   BOT_TOKEN=ваш_токен \
   WEBAPP_URL=https://ВАШ_ДОМЕН/webapp/index.html \
   PRO_PRICE_STARS=750 \
   node server.js
   ```
   Разместить можно на Railway, Render, VPS — где удобно (нужен публичный HTTPS-URL).
5. **Поставить вебхук:**
   ```bash
   curl "https://api.telegram.org/botВАШ_ТОКЕН/setWebhook?url=https://ВАШ_БЭКЕНД/webhook/ВАШ_ТОКЕН"
   ```
6. **Указать адрес бэкенда в Mini App.** В `webapp/app.js` поменяйте `BACKEND_URL` на адрес из шага 4.
7. **Включить приём Stars.** В @BotFather проверьте, что у бота разрешены платежи (Stars включены по умолчанию).

Готово: пользователь открывает бота → жмёт «Открыть калькулятор» → считает бесплатно →
жмёт Pro → платит Stars → функции открываются мгновенно.

## Что было исправлено при портировании из Excel

1. **Мёртвое поле «Из них с НДС 22%»** — в Excel собиралось, но нигде не использовалось.
   В приложении убрано из ввода (логика НДС 5%/7% сохранена и оформлена честно как упрощение).
2. **Мусорная ячейка** `madeofpoison` из листа «Ваши данные» — не переносится.
3. Вся логика покрыта тестами (`test/verify.js`) на эталонных цифрах из исходника.

---

## Обновление на следующий год

Поменялись ставки или лимиты? Правьте **только** `shared/params.js` — сайт, мини-апп и PDF
обновятся автоматически. После правок прогоните `node test/verify.js` и при необходимости
поправьте эталонные значения в тесте.
