# Публикация «Налогового навигатора» в Telegram — пошагово

Делается в два этапа. **Этап 1** (сайт + калькулятор в Telegram) — без своего сервера, работает бесплатный режим. **Этап 2** (оплата Pro) — нужен сервер, делается позже.

Проект уже подготовлен: создан git-репозиторий, секретов нет, корневой редирект на лендинг работает.

## ЭТАП 1. Сайт и Mini App на GitHub Pages (≈20 минут)

### Шаг 1. Создать аккаунт GitHub
Если нет — зарегистрируйтесь на https://github.com (бесплатно).
⚠️ Если GitHub не открывается — включите VPN (как для Claude Code).

### Шаг 2. Создать репозиторий
1. Нажмите **+** справа вверху → **New repository**.
2. **Repository name:** `tax-navigator` (или любое).
3. Оставьте **Public** (для бесплатного GitHub Pages нужен публичный).
4. НЕ ставьте галочки «Add README / .gitignore» — у нас уже всё есть.
5. **Create repository**.

### Шаг 3. Загрузить проект
GitHub покажет страницу с командами. Нам нужен блок **«…or push an existing repository»**. Скопируйте оттуда адрес репозитория (вида `https://github.com/ВАШ_ЛОГИН/tax-navigator.git`) и выполните в PowerShell:

```powershell
cd "C:\Users\XDot PC\OneDrive\Desktop\tax-navigator"
git remote add origin https://github.com/ВАШ_ЛОГИН/tax-navigator.git
git branch -M main
git push -u origin main
```
При запросе логина/пароля: вместо пароля GitHub требует **токен** (Personal Access Token). Как создать: GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate → отметьте `repo` → скопируйте и вставьте вместо пароля.

*(Проще: установить GitHub Desktop — https://desktop.github.com — и загрузить папку мышкой, без команд.)*

### Шаг 4. Включить GitHub Pages
1. В репозитории: **Settings** → слева **Pages**.
2. **Source:** Deploy from a branch.
3. **Branch:** `main`, папка `/ (root)` → **Save**.
4. Подождите 1–2 минуты. Вверху появится адрес:
   `https://ВАШ_ЛОГИН.github.io/tax-navigator/`

### Шаг 5. Проверить
Откройте этот адрес — должен открыться лендинг с калькулятором.
- Лендинг: `https://ВАШ_ЛОГИН.github.io/tax-navigator/`
- Mini App: `https://ВАШ_ЛОГИН.github.io/tax-navigator/webapp/index.html`
- Политика: `https://ВАШ_ЛОГИН.github.io/tax-navigator/landing/privacy.html`

**Запишите адрес Mini App — он нужен для бота.**

## ЭТАП 1Б. Создать бота и привязать Mini App (≈10 минут)

### Шаг 6. Создать бота
1. В Telegram напишите **@BotFather**.
2. `/newbot` → придумайте имя и username (должен заканчиваться на `bot`, например `tax_navigator_bot`).
3. BotFather пришлёт **токен** — сохраните его (понадобится на Этапе 2).

### Шаг 7. Поставить аватар и описание
В @BotFather:
- `/setuserpic` → выберите бота → загрузите `brand/bot-avatar-512.png`
- `/setdescription` → текст: «Сравните 6 налоговых режимов ИП с учётом реформы НДС 2026 и узнайте, сколько можно сэкономить. Бесплатный расчёт за минуту.»
- `/setabouttext` → «Налоговый навигатор ИП 2026»

### Шаг 8. Привязать Mini App
В @BotFather:
1. `/newapp` → выберите бота.
2. Название, описание, картинку (можно `brand/icon-512.png`).
3. **Web App URL:** адрес из шага 5 — `https://ВАШ_ЛОГИН.github.io/tax-navigator/webapp/index.html`
4. Готово — у бота появится кнопка запуска приложения.

### Шаг 9. Кнопка-меню (по желанию)
`/setmenubutton` → выберите бота → URL тот же → текст кнопки «🧮 Калькулятор».

## ЭТАП 2. Подключить оплату Pro (позже, нужен сервер)

Mini App сейчас проверяет Pro по ссылке `?pro=1` (заглушка). Для реальной оплаты:

1. **Завести сервер для бота** (`bot/server.js`). Бесплатные варианты: Railway, Render, Cloud.ru, любой VPS с Node 18+.
2. **Задать переменные** при запуске:
   ```
   BOT_TOKEN=токен_из_шага_6
   WEBAPP_URL=https://ВАШ_ЛОГИН.github.io/tax-navigator/webapp/index.html
   PRO_PRICE_STARS=750
   ```
3. **Поставить вебхук:**
   ```
   https://api.telegram.org/botТОКЕН/setWebhook?url=https://ВАШ_СЕРВЕР/webhook/ТОКЕН
   ```
4. **Прописать адрес сервера** в `webapp/app.js`: поменять `BACKEND_URL` и `PRIVACY_URL` на реальные.
5. Перезалить webapp на GitHub Pages (`git push`).

После этого кнопка «Оформить Pro» откроет оплату Telegram Stars, и после оплаты функции откроются автоматически.
