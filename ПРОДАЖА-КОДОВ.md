# Продажа Pro через коды доступа — инструкция

Самый быстрый способ продавать Pro **уже сейчас**, без сервера и эквайринга.

## Как это работает

1. У вас есть список кодов (файл `КОДЫ-ДОСТУПА.txt`, формат `TN26-XXXXX-XXXXX`).
2. Покупатель платит вам **любым способом** — перевод на карту, СБП, наличные, Авито.
3. Вы присылаете ему **один код**.
4. Он открывает мини-апп → жмёт «У меня есть код доступа» → вводит → **Pro открывается** и сохраняется на его устройстве.

Код проверяется прямо в приложении (без интернета к серверу) — подделать нельзя.

## Что делать вам

### Перед первой продажей (один раз)
1. **Поменяйте секрет.** Откройте `shared/codes.js`, в строке `CODES_SECRET` замените значение на свою длинную случайную строку. ⚠️ Это аннулирует коды из примера — генерируйте новые ПОСЛЕ смены секрета.
2. **Сгенерируйте свои коды:**
   ```powershell
   cd "C:\Users\XDot PC\OneDrive\Desktop\tax-navigator"
   node bot/generate-codes.js 50 > КОДЫ-ДОСТУПА.txt
   ```
   (50 — сколько кодов нужно)
3. **Запушьте смену секрета на сайт:**
   ```powershell
   git add -A
   git commit -m "Свой секрет кодов"
   git push
   ```

### При каждой продаже
1. Получили оплату → откройте `КОДЫ-ДОСТУПА.txt`.
2. Возьмите **неиспользованный** код, отправьте покупателю.
3. **Отметьте его как использованный** (поставьте рядом «✓ Иванов, 03.06») — чтобы не выдать дважды.

## Важно понимать (честно про ограничения)

- **Код работает на устройстве покупателя** и там запоминается. Если он переустановит Telegram — код нужно ввести снова (он его знает).
- **«Одноразовость» вы контролируете списком**, а не программой (сервера пока нет). Один код = одна продажа. Не публикуйте коды списком — раздавайте поштучно.
- Если кто-то поделится своим кодом с другом — технически тот тоже активирует. Для старта это приемлемо (как с лицензионным ключом). Когда продаж станет много — подключим сервер с настоящей одноразовостью (Этап 2).

## Сколько брать

Цена та же — **1990 ₽ разово** (или назначайте свою). Через коды вся сумма ваша, без комиссии Telegram.

## Когда надоест выдавать вручную

Подключим **Продамус/ЮKassa**: кнопка «Купить» → оплата картой с чеком → код приходит автоматически. Это следующий шаг, когда поток продаж станет регулярным.
