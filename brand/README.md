# Брендовый комплект — Налоговый навигатор ИП 2026

Фирменный стиль: индиго `#6366f1` → фиолетовый `#a855f7`, знак «растущий график со стрелкой».
Шрифты логотипа: **Jura** (название), **IBM Plex Mono** (подпись).

## Файлы и для чего они

| Файл | Размер | Где использовать |
|---|---|---|
| `icon-512.png` | 512×512 | Основная иконка-знак (плашка с градиентом) |
| `icon-192.png` | 192×192 | Иконка поменьше (PWA, apple-touch-icon) |
| `favicon-64.png` | 64×64 | Фавикон сайта (вкладка браузера) |
| `icon-white-transparent.png` | 512×512 | Белый знак на прозрачном — для печати на тёмном/цветном |
| `bot-avatar-512.png` | 512×512 | **Аватар Telegram-бота** (загрузить в @BotFather → /setuserpic) |
| `logo-horizontal-light.png` | 1400×420 | Полный логотип на светлом фоне (документы, счета) |
| `logo-horizontal-dark.png` | 1400×420 | Полный логотип на тёмном фоне (презентации) |
| `logo-horizontal-on-dark.png` | 1400×420 | Прозрачный фон, светлый текст — кладётся на тёмные фоны |
| `logo-horizontal-on-light.png` | 1400×420 | Прозрачный фон, тёмный текст — кладётся на светлые фоны |

## Как пересоздать
`python make_final.py` — перерисует весь комплект (например, если поменяете цвета бренда вверху скрипта).

## Уже подключено
- Фавикон и og:image сайта → `landing/assets/` (копии favicon-64, icon-192/512, logo-horizontal-on-dark).
- Цвет темы сайта обновлён на `#6366f1`.

## Что осталось сделать вам при публикации
- Аватар бота: @BotFather → `/setuserpic` → загрузить `bot-avatar-512.png`.
