## 6. Сборка кампании: тип, оптимизация, разметка (читать перед запуском)

**Тип кампании.** На старте — **«Единая перформанс-кампания» (ЕПК) → место показа «Поиск»**, с ручной
структурой групп из раздела 1. «Мастер кампаний» проще, но любит сам подмешивать РСЯ и автотаргетинг
и жечь бюджет на нерелевантном — для горячего intent'а («какой режим выбрать») начинаем чисто с Поиска.

**На какую цель оптимизировать.**
- **Неделя 1** — стратегия **«Максимум кликов»** с ограничением ставки/недельным бюджетом (или ручное
  управление). Конверсий ещё мало, автостратегии по цели пока не на чем учиться — сначала копим данные
  и чистим минус-слова.
- **Дальше** — переключить на **«Максимум конверсий» по цели `open_bot`** (нужно ~10 конверсий/нед,
  чтобы стратегия работала). Это заставит Директ искать тех, кто реально переходит в бота, а не просто кликает.

**⚠️ Разметка ссылок для Метрики — ВКЛЮЧИТЬ.** В параметрах кампании включи
«**Размечать ссылки для Метрики**» (Директ добавит `yclid`). Без этого Метрика не свяжет переход в бота
с конкретной кампанией/группой/объявлением — не увидишь, какая группа даёт `open_bot` дешевле.
Счётчик `109693939` тоже привязать к кампании в её настройках.

**Автотаргетинг** — на старте сузить до «целевых запросов» или выключить (иначе показы по левым фразам).
**РСЯ (сети)** — на старте выключить, только Поиск. Расширишь позже, когда найдёшь рабочие связки.
**Корректировки** — минус 100% на возраст до 18.

**Реальность по цене клика.** Ниша конкурентная: по «калькулятор налогов ип», «усн», «какой режим»
бьются банки (Точка, Тинькофф, Альфа), Эльба/Контур, «Моё дело». Клик по горячим ключам реально
**30–120 ₽**. При 500 ₽/день это ~5–15 кликов/день — это нормально для теста. Задача первой недели —
не объём, а найти **1–2 группы/ключа с самым дешёвым «переходом в бота»** (по отчёту Метрики на цель
`open_bot`) и потом лить бюджет туда.
